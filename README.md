# Minimalistic-GTNH-repair
极简主义修复版，目前修复仍在进行中。本意是打造一个相对干净整齐的16x材质包。

目前持续更新修复的材质有：Gregtech和一些附属；IndustrialCraft2；Ender IO。

附赠额外mod材质：[BetterTooltipBox](https://github.com/xiaoxing2005/BetterTooltipBox)提示框皮肤。
